# DittNAV proxy

Proxy mellom dittnav-frontend og dittnav-api/dittnav-event-handler

# Kom i gang
1. Bygg dittnav-proxy ved å kjøre `gradle build`
2. Start appen ved å kjøre filen App.kt.
