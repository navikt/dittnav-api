# DittNAV api

Proxy mellom dittnav-frontend og dittnav-legacy-api/dittnav-event-handler

# Kom i gang
1. Bygg dittnav-api ved å kjøre `gradle build`
2. Start appen lokalt ved å kjøre main-funksjonen i filen TestApp.kt.
